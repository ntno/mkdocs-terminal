# mkdocs-terminal
[Gioni06/terminal.css](https://github.com/Gioni06/terminal.css) theme for MkDocs

## Quick start

terminal.css theme for MkDocs can be installed with `pip`:

``` sh
pip install mkdocs-terminal
```

Add the following lines to `mkdocs.yml`:

``` yaml
theme:
  name: terminal
```
